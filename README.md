# Digital-literacy-project
 The project demonstrates my understanding of digital tools, online professionalism, cybersecurity awareness, and responsible digital communication through five practical tasks.
Module Summaries
Task 1 – Digital Literacy Infographic

Created an infographic explaining digital literacy, useful student tools, safe internet practices, and email etiquette using PowerPoint.

Task 2 – Student Digital Portfolio

Set up professional profiles on platforms like GitHub and LinkedIn to begin building an academic and professional online presence.

Task 3 – Coding & Collaboration Platforms

Completed a beginner coding challenge and created a Google Form quiz to demonstrate collaboration tools.

Task 4 – Email Etiquette

Drafted professional emails and created a checklist of social media do's and don'ts for responsible digital communication.

Task 5 – Cybercrime Awareness

Researched a cybercrime case study and created a prevention checklist with safety tips for students.
